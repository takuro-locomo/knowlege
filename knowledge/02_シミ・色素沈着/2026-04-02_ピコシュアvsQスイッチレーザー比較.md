# ピコシュア vs Qスイッチレーザー（ナノ秒）比較

## 【ファクト】

- **パルス幅の違い**：
  - ナノ秒レーザー（Qスイッチ）：1〜100ナノ秒（10億分の1秒）
  - ピコ秒レーザー（ピコシュア等）：数百ピコ秒（1兆分の1秒）
  → ピコ秒はナノ秒の**約1000分の1**の超短パルス。
- **作用の違い**：
  - ナノ秒：光熱作用（photothermal）メインでメラニンを熱破壊
  - ピコ秒：光音響作用（photoacoustic）メインで衝撃波により粉砕 → **周囲組織への熱拡散が少ない**
- **PIH（炎症後色素沈着）リスク比較**：
  - ピコ秒レーザーは、ナノ秒レーザーやIPLと比較して**PIHリスクを最大60%低減**するとされる（特にFitzpatrickⅢ〜Ⅵの濃い肌色に有利）。
  - 日本人・アジア人のようにメラニン量が多い肌色（Ⅲ〜Ⅳ）では、ナノ秒よりピコ秒の安全性が高い。
- **シミへの効果比較**（PMC・Frontiers論文より）：
  - 老人性色素斑・そばかす：両者ともに有効。ピコ秒の方がダウンタイム短縮傾向。
  - 肝斑：高出力のQスイッチは禁忌（悪化リスク）。低出力ピコトーニングが推奨。
  - 真皮性色素沈着（ADM・太田母斑）：ピコ秒が優位（より細かく粉砕できるため）。
- **フラクショナルモードはピコ秒のみの強み**：Qスイッチには相当するフラクショナルモードがなく、毛穴・ニキビ跡・肌質改善はピコ秒の独自領域。
- **コスト面**：Qスイッチレーザーは機器コストが低く施術費も低めになる傾向。ピコシュアは高性能な分、費用が高くなる場合が多い。
- **まとめの使い分け指針**：

  | シミの種類 | 推奨 |
  |-----------|------|
  | 老人性色素斑・そばかす | ピコスポット（または Qスイッチ） |
  | 肝斑 | ピコトーニング（Qスイッチ高出力は禁忌） |
  | ADM・真皮性色素 | ピコシュア優位 |
  | 毛穴・ニキビ跡・肌質 | ピコフラクショナル（Qスイッチは不向き） |
  | 色黒・アジア人肌 | ピコシュア優位（PIHリスク低） |
